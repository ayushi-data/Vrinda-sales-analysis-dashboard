\## Project Overview
This project analyzes Vrinda Store sales data using an Excel dashboard to track performance and customer purchasing trends.

## Tools Used
- Microsoft Excel

## Key KPIs
- Total Sales
- Total Orders
- Order Status Analysis
- Sales by Channel
- Customer Demographics

## Insights
- Majority of orders are delivered successfully
- Specific sales channels contribute higher revenue
- Certain customer segments show stronger purchasing behavior
